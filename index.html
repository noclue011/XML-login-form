Since modern web browsers do not allow javascript to access local xml files because of the CORS protocol, this application must be run either from a local server or from a browser that is set up to allow local files. In order for the login functionality to work:

1) open the command console (if using Windows)
2) use the 'cd' command to navigate to the folder with chrome.exe
3) launch the Google Chrome browser from the command console with the additional argument '--allow-file-access-from-files'


This will allow the javascript code to access the local xml file in the project folder.

This program can also be run from a local server to allow the program to run correctly.

If neither of these solutions are used then the login page won't work since the CORS protocol will block the program from accessing the xml file.
